# 个人作品集网站

这是一个展示我个人技能和项目的响应式网站，专为大学生开发者设计。

## 功能特点

- 响应式设计，适配各种设备
- 3D粒子交互背景（青蓝色和深蓝色渐变）
- 四个独立页面：首页、动态、团队、关于我
- 现代化UI设计
- 平滑的动画效果

## 技术栈

- HTML5
- CSS3 (Flexbox, Grid, 媒体查询)
- JavaScript (原生JS + Particles.js)
- Font Awesome图标

## 页面说明

### 首页 (index.html)
- 个人介绍和技能展示
- 主要项目亮点
- 快速导航到其他页面

### 动态页面 (blog.html)
- 技术博客文章
- 学习笔记和心得
- 项目更新日志

### 团队页面 (team.html)
- 团队成员介绍
- 合作项目展示
- 团队成就

### 关于我页面 (about.html)
- 详细个人经历
- 教育背景
- 技能展示
- 联系方式

## 本地开发

1. 克隆或下载项目文件
2. 直接在浏览器中打开index.html文件
3. 或者使用Live Server等本地服务器

## 自定义指南

### 修改内容
- 直接编辑HTML文件中的文本内容
- 更新技能部分反映您的实际技能

### 修改样式
- 主要样式写在css/style.css文件中
- 粒子效果样式在css/particles.css中
- 修改颜色主题：更新CSS中的颜色值

### 修改粒子效果
- 在js/particles.js中调整particlesJS配置参数
- 参考particles.js文档获取更多配置选项

## 部署

可以直接部署到任何静态网站托管服务：
- GitHub Pages
- Netlify
- Vercel
- 或其他传统虚拟主机

## 浏览器支持

支持所有现代浏览器：
- Chrome (推荐)
